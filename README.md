# 📂 HWP/PDF → Markdown 일괄 변환기 (macOS & Windows 지원)

한글 문서(.hwp) 및 PDF 문서(.pdf)들을 일괄적으로 마크다운(.md) 양식으로 깨끗하게 변환해 주는 로컬 웹 애플리케이션입니다. Obsidian 보관소 등으로 손쉽게 복사하거나 동기화하여 활용할 수 있습니다.

특히 목회자 설교문이나 성경 묵상 텍스트에 특화된 포맷터와 인공지능 요약 기능이 장착되어 있습니다.

---

## ✨ 주요 특징

### 1. 3단계 강력한 HWP Fallback 변환 엔진
*   **1단계 (`hwp5txt`)**: `pyhwp` 라이브러리를 사용하여 원본 문서 레이아웃과 텍스트를 가장 완벽하게 추출합니다.
*   **2단계 (`olefile` 직접 파싱)**: 1단계 실패 시 OLE 바이너리 구조를 직접 압축 해제(`zlib`) 및 파싱하여 본문 텍스트를 끝까지 복원합니다.
*   **3단계 (Python strings 파서)**: 최후의 보루로서 2바이트 UTF-16LE 바이트 스트림을 한 글자씩 스캔하여 한글 텍스트만 안전하게 추출합니다.

### 2. 하이브리드 PDF 텍스트 추출 및 지능형 OCR
*   **텍스트 직접 추출**: `pypdf`를 사용하여 PDF 내부에 내장된 텍스트 스트림을 고속 추출합니다.
*   **글자 깨짐 감지 (한글 폰트 매핑 오류 대응)**: PDF 텍스트를 추출했을 때 폰트 매핑 오류로 인한 특수문자나 깨짐 현상(PUA 문자, 유니코드 깨짐 등)이 발생하면 시스템이 이를 자동으로 감지하여 OCR 모드로 자동 전환합니다.
*   **macOS Vision OCR (기본 탑재)**: Apple Silicon 및 Intel macOS 환경인 경우, 애플의 고성능 온디바이스 `Vision` 프레임워크를 활용해 별도의 클라우드 트래픽이나 비용 없이 오프라인 상태에서 완전 무료로 최고 성능의 한글/영어 OCR을 수행합니다.
*   **Gemini OCR Fallback**: 스캔된 이미지 PDF이거나 텍스트 추출이 불가능할 때, Google Gemini AI 비전 모델을 사용하여 정밀한 이미지 텍스트 추출을 수행합니다.

### 3. Google Gemini AI 연동 (API 키 및 구독 연동 지원)
*   **무료 API Key 방식**: 개인 Gemini API Key를 입력하여 요약 기능을 활성화할 수 있습니다.
*   **Google 계정 구독 연동 (API Key 불필요) [NEW]**:
    *   Gemini Advanced 등 유료 구독을 사용 중인 Google 계정이 있다면, **별도의 API Key 발급 없이 구글 로그인 연결(OAuth 2.0 PKCE)**만으로 고성능 Gemini 모델을 이용할 수 있습니다.
    *   로그인을 완료하면 백엔드 서버에서 백그라운드 콜백 수신기(포트 `8085`)를 가동하여 토큰을 교환하고, 자동으로 안전한 클라우드 AI 에이전트 환경(Cloud Code Assist)을 프로비저닝 및 연동합니다.

### 4. 목회/설교 특화 커스텀 마크다운 변환 규칙
*   **성경 구절 인용구 변환**: 한글 성경 약어 패턴(예: `창 1:1`, `마 3장 16절` 등)을 감지하여 옵시디언 인용블록(`> `)으로 자동 변경합니다.
*   **문단 제목(Heading) 생성**: 30자 미만의 짧은 문장 중 마침표(`.`)나 한국어 문장 결미어(`다`, `요`, `니다` 등)로 끝나지 않는 줄을 `## 제목`으로 자동 승격합니다.
*   **목록(List) 기호 일치**: `•`, `·`, `▶` 등의 특수 목록 기호를 마크다운 표준 목록인 `- `로 자동 정렬합니다.
*   **빈 줄 정리**: 가독성을 위해 본문 내에 불필요하게 겹쳐진 연속된 빈 줄을 1개의 빈 줄로 압축합니다.
*   **Frontmatter 메타데이터 & Callouts**: Gemini 분석 활성화 시 설교 제목, 성경 구절, 핵심 주제, 키워드, AI 생성 10대 태그 정보가 YAML Frontmatter 및 🔑 3대 핵심 요약/📖 전체 요약 옵시디언 Callout으로 자동 포맷팅되어 추가됩니다.

## 💡 AI 및 구독 연동 시 차별화된 장점 (활용 가치)

Gemini API Key 또는 Google 구독을 연동하여 AI 기능을 활성화하면, 단순한 **'텍스트 파싱기'**에서 옵시디언 환경에 최적화된 **'지능형 지식 아카이빙 비서'**로 역할이 완전히 업그레이드됩니다.

### 1. 옵시디언(Obsidian) 검색 및 자동 태깅 최적화
*   **AI 미연동**: 원본 텍스트만 추출되므로, 나중에 원하는 주제나 설교를 찾으려면 본문 전체를 일일이 키워드로 검색해야 하여 정확도가 떨어집니다.
*   **AI 연동 시**: Gemini가 설교의 영적 맥락과 주제를 완전하게 이해하고 핵심 키워드 및 추천 해시태그 10개를 자동 생성합니다. 옵시디언 내에서 `#창세기`, `#은혜`, `#위로` 등의 태그 클릭 한 번으로 관련 설교 및 묵상을 완벽하게 연계·분류할 수 있습니다.

### 2. 설교 준비 및 아카이빙 생산성 극대화 (미리보기 비서)
*   **AI 미연동**: 과거 노트를 참고하려면 수천 단어의 본문 전체를 처음부터 끝까지 다 읽어보아야 내용을 파악할 수 있습니다.
*   **AI 연동 시**: 마크다운 파일 최상단에 **🔑 3가지 핵심 요약 콜아웃**과 **📖 전체 요약문**이 수록됩니다. 노트를 열자마자 10초 만에 과거 설교의 핵심 뼈대와 핵심 내용을 파악할 수 있어 설교 참고 및 준비 시간이 획기적으로 줄어듭니다.

### 3. 글자가 깨진 PDF 및 이미지 스캔본 심폐소생 (강력한 복구)
*   **AI 미연동**: 한글 파일에서 PDF로 내보낼 때 폰트 매핑이 꼬인 파일이나, 종이 설교 인쇄물을 스캔한 PDF는 변환 시 빈 문서가 되거나 깨진 외계어(예: `최 足오- 혀(오☜` 등)로 출력됩니다.
*   **AI 연동 시**: 시스템이 본문 내 한글 깨짐을 자동으로 수치 분석하여, 오동작 감지 시 즉각 **페이지 단위의 AI 스크샷 OCR**을 실행합니다. 이를 통해 깨진 문서를 완전히 정갈한 한글/영문 마크다운 문서로 새롭게 복구해 냅니다.

### 4. 구글 구독 연동 방식만의 특장점 (OAuth PKCE)
*   **과금 걱정 제로**: 매달 사용량에 따라 카드 결제가 청구되는 API Key 방식과 달리, 사용 중인 **Gemini Advanced 등 개인 구글 유료 구독 용량**을 활용하므로 추가 비용 부담 없이 대량의 문서를 무료로 변환할 수 있습니다.
*   **간편한 보안 관리**: 유출 우려가 있는 긴 영문/숫자 형태의 API Key를 복사 및 노출할 필요 없이, 브라우저 구글 로그인 클릭 한 번으로 안전하게 승인 및 연동을 완료합니다.

## ⚙️ 실행 전 준비사항 (최초 1회)

이 프로그램은 **파이썬(Python 3)**을 사용하여 작동합니다. 컴퓨터에 파이썬이 설치되어 있지 않다면 먼저 설치해 주세요.

*   [Python 3 공식 다운로드 페이지](https://www.python.org/downloads/)
*   **Windows 사용자**: 설치 창 하단의 **[Add Python to PATH]** 체크박스를 반드시 체크하고 설치하셔야 합니다.
*   **macOS 사용자**: 터미널에서 `python3`을 입력하시거나 실행기를 실행할 때 Xcode 명령줄 도구 설치 팝업이 나타날 수 있으니 승인하여 설치해 주시면 됩니다.

## 🚀 사용 방법

### 1. 다운로드 및 압축 해제
이 저장소 상단의 **[Code]** 초록색 버튼 클릭 -> **[Download ZIP]**을 눌러 다운로드한 뒤 원하는 폴더에 압축을 풉니다.

### 2. 더블클릭 실행
자신의 운영체제(OS)에 맞는 실행 파일을 **더블클릭**합니다.
*   **macOS**: `run.command` 파일을 더블클릭합니다.
    > 💡 **macOS 최초 실행 시 보안 경고 해결법**
    > 인터넷에서 직접 받은 파일이므로 macOS 보안 정책에 의해 실행이 차단될 수 있습니다.
    > 1. `run.command` 파일 위에서 마우스 **우클릭** (또는 트랙패드 두 손가락 클릭)
    > 2. **[열기 (Open)]** 메뉴 선택
    > 3. 경고창이 나타나면 다시 한번 **[열기]** 버튼 클릭
*   **Windows**: `run.bat` 파일을 더블클릭합니다.

자동으로 터미널/명령 프롬프트 창이 열리며, 변환에 필요한 파이썬 모듈(`Flask`, `pyhwp`, `olefile` 등)을 최초 1회 다운로드하고 **로컬 서버(http://localhost:5001)가 즉시 열립니다.**

### 3. 변환기 조작
1.  **[한글 파일(HWP) 폴더 경로]**: 변환할 한글 파일들이 보관된 컴퓨터의 절대 경로를 입력합니다.
2.  **[Obsidian 보관소(Vault) 저장 경로]**: 변환 결과물(.md)을 저장할 옵시디언 폴더의 절대 경로를 지정합니다.
3.  **[Google 계정 연결]** 또는 **[Gemini API Key]**를 설정하여 AI 요약 및 텍스트 보정 옵션을 켭니다.
4.  **[파일 목록 확인]** 버튼을 눌러 스캔된 파일의 개수와 리스트를 확인합니다.
5.  **[변환 시작]**을 누르면 변환이 진행되며 결과 리포트를 받아볼 수 있습니다.

### 4. 종료 방법
실행 중인 검은색 터미널/명령 프롬프트 창을 닫거나, 창에서 **`Ctrl + C`**를 누르면 안전하게 백엔드 서버가 종료됩니다.

