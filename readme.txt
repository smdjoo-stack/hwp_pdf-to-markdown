📂 HWP/PDF → Markdown 일괄 변환 웹 애플리케이션 사용 설명서
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

이 프로그램은 한글 파일(HWP) 및 PDF 파일들을 마크다운(Markdown) 문서로 일괄 변환하여
저장해주는 로컬 웹 앱입니다. (Obsidian 보관소 등으로 쉽게 가져갈 수 있습니다.)

--------------------------------------------------
📌 1. 실행 전 준비사항 (최초 1회만 필요)
--------------------------------------------------
이 프로그램은 파이썬(Python 3)을 기반으로 실행됩니다.
만약 컴퓨터에 파이썬이 설치되어 있지 않다면 아래 공식 사이트에서 다운로드 후 설치해주세요.
▶ 파이썬 다운로드: https://www.python.org/downloads/
(※ 윈도우 사용자는 설치 창 하단의 'Add Python to PATH' 옵션을 꼭 체크해주세요.)
(※ macOS는 기본적으로 파이썬이 탑재되어 있거나 터미널 실행 시 설치 안내가 나타납니다.)

--------------------------------------------------
🚀 2. 실행 방법 (더블클릭 실행)
--------------------------------------------------
- macOS: 이 폴더 안에 있는 'run.command' 파일을 마우스로 더블클릭합니다.
- Windows: 이 폴더 안에 있는 'run.bat' 파일을 마우스로 더블클릭합니다.

검은색 창이 열리면서 실행에 필요한 최신 패키지(Flask, pyhwp, pypdf 등)를 자동으로 안전하게 설치합니다. (첫 실행 시 10~30초 소요)
설치 및 준비가 끝나면 웹 브라우저(http://localhost:5001)에 변환기 화면이 자동으로 열립니다.

※ macOS 보안 경고가 발생하는 경우:
인터넷에서 받은 파일이라 실행이 차단되는 경우, 'run.command' 파일 위에서 마우스 우클릭(또는 패드 두 손가락 클릭) -> [열기(Open)]를 선택한 다음 나타나는 창에서 [열기]를 클릭해주시면 정상 실행됩니다.

--------------------------------------------------
💻 3. 변환기 사용 방법
--------------------------------------------------
1. [문서 파일(HWP/PDF) 폴더 경로]: 변환하려는 HWP 또는 PDF 파일들이 모여있는 폴더 경로를 입력합니다.
2. [Obsidian 보관소(Vault) 저장 경로]: 변환된 마크다운(.md) 파일이 저장될 옵시디언 폴더 경로를 입력합니다. (기본값 설정되어 있음)
3. [파일 목록 확인]: 경로 입력 후 버튼을 누르면 변환할 파일들이 몇 개인지 미리 스캔하여 목록을 보여줍니다.
4. [변환 시작]: 보라색 변환 시작 버튼을 누르면 변환 진행률과 함께 각 파일의 세부 변환 상태가 실시간으로 기록됩니다.
5. [Obsidian에서 열기]: 변환이 완료된 후 버튼을 누르면 해당 옵시디언 폴더가 Finder 또는 탐색기 창으로 열립니다.

--------------------------------------------------
❌ 4. 종료 방법
--------------------------------------------------
실행 중인 터미널/명령 프롬프트 창을 닫거나, 활성화 상태에서 키보드의 [Ctrl + C]를 누르면 서버가 즉시 안전하게 종료됩니다.

--------------------------------------------------
📝 5. 변환 규칙 및 기타
--------------------------------------------------
- 변환된 모든 문서 상단에는 파일명, 원본 파일명, 변환 날짜가 옵시디언 Frontmatter 양식으로 기록됩니다. (PDF는 pdf-converted 태그가, HWP는 hwp-converted 태그가 지정됩니다.)
- 문서 내의 성경 본문 인용(예: 창 1장, 요 3:16)은 옵시디언 인용구(> ) 형태로 자동 변환됩니다.
- 30자 이하의 짧은 줄(마침표 등으로 끝나지 않는 줄)은 제목(## Heading)으로 자동 변환됩니다.
- 특수문자 기호(•, ·, ▶ 등)는 리스트(- 리스트) 형태로 변형됩니다.
- 변환 중 일부 손상된 파일이 있어도 중단되지 않고 패스되며, 변환 실패 이력은 자동으로 생성되는 'conversion_errors.log' 파일에서 시간과 사유를 상세하게 확인하실 수 있습니다.
